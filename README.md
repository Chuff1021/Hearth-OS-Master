# Hearth-OS Master Demo Workspace

This repo is the safe demo workspace for the combined Hearth-OS + Fireplace Superstore concept.

It intentionally keeps the production repos untouched:

- `apps/hearth-os-demo` — demo copy of Hearth-OS, using test/demo data only.
- `apps/superstore-demo` — demo copy of the Fireplace Superstore ecommerce site.

## Demo rollout order

1. Travis Industries / Fireplace Xtrordinair / Lopi demo
2. Hearth & Home Technologies demo
3. Superior / IHP demo
4. Additional manufacturer/dealer demos

## Goal

Show manufacturers and dealers a complete dealer operating platform:

- white-labeled dealer ecommerce superstore
- product/model/SKU/parts catalog already loaded
- online quote/order/service request intake
- Hearth-OS dashboard receiving website orders and service requests
- test-data workflows for customers, jobs, invoices, payments, parts, and reporting

## Important safety rule

Do not use production credentials, production customer data, or live payment credentials in this repo. This workspace is for demos only.
