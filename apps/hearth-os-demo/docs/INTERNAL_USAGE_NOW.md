# HearthOS / GABE — Start Using This Now

## Live tools
- Manuals: `/tech/manuals`
- GABE: `/tech/gabe`

## How staff should use it (simple flow)
1. Open `/tech/manuals` and confirm the manual/model context.
2. Ask the question in `/tech/gabe`.
3. Verify the response includes manual citation context when applicable.
4. If something is wrong/unclear, submit feedback immediately using the feedback path below.

## Feedback path (required categories)
Submit one feedback item per issue with:
- timestamp
- user
- page (`/tech/manuals` or `/tech/gabe`)
- model/manual context
- question asked
- answer received
- issue type (pick one):
  - wrong answer
  - missing manual
  - unclear citation
  - unsupported question
- short note

## Read-only safety
- No destructive actions.
- No ordering or fitment confirmation workflows here.
- Treat output as decision support; verify model/serial context before action.
